Support

LeetBeam is a Chrome extension for tracking accepted LeetCode submissions and reviewing progress.

Common help

Reload the extension after updates, refresh the LeetCode page before resyncing, and verify optional AI settings if AI coaching is enabled.

Troubleshooting

If progress sync finds no rows, refresh the LeetCode progress page and try again. If a tracked entry is wrong, delete the entry or clear imported history and resync.
