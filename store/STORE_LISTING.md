# Chrome Web Store Listing Copy

## Title

LeetBeam

## Short Summary

Track accepted LeetCode solves, review daily progress, and get local-first interview coaching.

## Detailed Description

LeetBeam helps you turn practice sessions into something you can actually review.

The extension tracks accepted LeetCode problems, stores progress locally in Chrome, and gives you a cleaner way to understand what you solved, what patterns you’re practicing, and where you should focus next.

### What it does

- Tracks accepted problems on LeetCode problem pages
- Syncs visible solve history from your LeetCode progress page, with submissions as a fallback
- Shows total solved tracked by the extension and current streak
- Displays recent accepted problems with local feedback
- Generates daily review and interview-coach summaries from deterministic analytics
- Supports optional AI narrative coaching if you add your own OpenAI API key

### Why it’s useful

- Local-first by default
- Still useful with no AI enabled
- Keeps core progress math deterministic
- Helps you practice more intentionally for interviews

### Important notes

- This extension relies on visible LeetCode page content and DOM structure
- It does not use private LeetCode APIs
- Optional AI coaching is disabled by default

## Single Purpose Statement

This extension tracks a user’s LeetCode problem-solving progress and provides review and coaching features based on that activity.
