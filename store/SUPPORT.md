## User-Facing Help Copy

LeetBeam is a local-first Chrome extension for tracking accepted LeetCode submissions and reviewing practice progress.

If you need help:

- Make sure the extension is reloaded after updates
- Refresh the LeetCode page before resyncing
- Use the `Open progress page` and `Resync from progress page` controls in the popup
- If optional AI coaching is enabled, verify that your API key and model settings are correct

## Known Troubleshooting Notes

- If progress sync finds no rows, refresh the LeetCode progress page and try again
- If a tracked submission is wrong, delete the entry or clear imported history and resync
- If AI coaching fails, check the saved API key and model name first
