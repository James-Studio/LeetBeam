# Changelog

## 0.1.0 - 2026-04-21

Initial LeetBeam release candidate.

### Added

- Manifest V3 Chrome extension structure
- Accepted submission capture from LeetCode problem pages
- Local storage for tracked submissions and aggregates
- Daily review and interview coach sections
- Rule-based feedback engine
- Progress page and submissions-page history sync
- Recent accepted list with delete controls
- Optional OpenAI-based narrative coaching
- AI settings page
- Publication-prep assets including icons, privacy policy draft, store listing copy, screenshot captions, support template, and packaging script

### Notes

- Core analytics remain deterministic and local
- AI coaching is optional and additive only
